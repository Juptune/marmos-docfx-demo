About

Contains the contents of a response line.

Notes

This struct contains a pinned slice of the Http1Reader's buffer, so after processing the response line, the user code must release the slice by destroying the struct (e.g. setting it to .init , letting it go out of scope, etc.)

In order to make it as clear as possible that data from this struct must not** be escaped, the access method is provided which enforces via the type system that string data is not escaped.
