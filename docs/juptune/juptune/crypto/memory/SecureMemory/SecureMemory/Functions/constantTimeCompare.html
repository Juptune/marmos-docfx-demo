constantTimeCompare(other) 

bool constantTimeCompare( const scope void[] other ) @trusted

Compares this memory to another memory in constant time for the given length of this memory.

Assertions

other must have the same length as this memory. There is no valid usecase for comparing secure memory of different lengths.

Notes

You should always use this function if other contains sensitive data. Failure to do so opens you up to timing attacks.

Params

other void[]

The memory to compare to.

Returns

true if the memory is equal, false otherwise.
