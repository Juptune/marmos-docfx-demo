encrypt(plaintext, additionalData, outCipherText, outCipherTextSlice) 

Result encrypt(
  SecureMemory plaintext,
  <bug: unknown> additionalData,
  <bug: unknown> outCipherText,
  <bug: unknown> outCipherTextSlice
)

Encrypts the given plaintext into the outCipherText buffer, using the given additionalData to calculate the authentication tag, and returns a slice outCipherText which contains the final encrypted ciphertext + authentication tag.

Assertions

plaintext must not be empty, it makes no sense to encrypt an empty buffer.

outCipherText must be at least plaintext.length + AeadAlgorithmT.ABYTES in length. Where ABYTES is typically the length of the authentication tag.

Notes

This is a "combined" AEAD encryption, where the authentication tag is appended to the ciphertext. You can retrieve the tag by taking the last AeadAlgorithmT.ABYTES bytes of outCipherTextSlice .

Params

plaintext SecureMemory

The plaintext to encrypt.

additionalData <bug: unknown>

The additional data to use when calculating the authentication tag.

outCipherText <bug: unknown>

The buffer to store the encrypted ciphertext + authentication tag in.

outCipherTextSlice <bug: unknown>

The slice of outCipherText that contains the encrypted ciphertext + authentication tag.

Throws

Anything that AeadAlgorithmT.encrypt throws.

Returns

A Result that indicates if the encryption was successful or not.
