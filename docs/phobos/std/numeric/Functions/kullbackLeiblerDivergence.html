Computes the $(LINK2 https://en.wikipedia.org/wiki/Kullback%E2%80%93Leibler_divergence, Kullback-Leibler divergence) between input ranges a and b , which is the sum $(D ai log(ai / bi)). The base of logarithm is 2. The ranges are assumed to contain elements in $(D [0, 1]). Usually the ranges are normalized probability distributions, but this is not required or checked by $(D kullbackLeiblerDivergence). If any element bi is zero and the corresponding element ai nonzero, returns infinity. (Otherwise, if $(D ai == 0 && bi == 0), the term $(D ai log(ai / bi)) is considered zero.) If the inputs are normalized, the result is positive.
