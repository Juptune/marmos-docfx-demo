  bool isFile() @property scope
  // Returns whether the file represented by this `DirEntry` is a symbolic link. 
  bool isSymlink() @property scope

  uint linkAttributes() @property scope

  ulong size() @property scope

  stat_t statBuf() @property scope
  // $(BLUE This function is Windows-Only.) 
  SysTime timeCreated() @property scope
  // Returns the time that the file represented by this `DirEntry` was last accessed. 
  SysTime timeLastAccessed() @property scope
  // Returns the time that the file represented by this `DirEntry` was last modified. 
  SysTime timeLastModified() @property scope
  // $(BLUE This function is POSIX-Only.) 
  SysTime timeStatusChanged() @property scope
}

About

Info on a file, similar to what you'd get from stat on a POSIX system.
