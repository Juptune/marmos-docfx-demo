About

Hook that reserves a special value as a "Not a Number" representative. For signed integrals, the reserved value is T.min . For signed integrals, the reserved value is T.max .

The default value of a $(D Checked!(X, WithNaN)) is its NaN value, so care must be taken that all variables are explicitly initialized. Any arithmetic and logic operation involving at least on NaN becomes NaN itself. All of $(D a == b), $(D a < b), $(D a > b), $(D a <= b), $(D a >= b) yield false if at least one of a and b is NaN.
