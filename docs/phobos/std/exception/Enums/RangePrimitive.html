About

This enum is used to select the primitives of the range to handle by the $(LREF handle) range wrapper. The values of the enum can be OR 'd to select multiple primitives to be handled.

RangePrimitive.access is a shortcut for the access primitives; front , back and opIndex .

RangePrimitive.pop is a shortcut for the mutating primitives; popFront and popBack .
