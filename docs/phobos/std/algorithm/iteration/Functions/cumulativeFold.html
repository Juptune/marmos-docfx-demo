cumulativeFold(range) 

auto cumulativeFold( R )( R range )

Similar to fold , but returns a range containing the successive reduced values. The call cumulativeFold!(fun)(range, seed) first assigns seed to an internal variable result , also called the accumulator. The returned range contains the values result = fun(result, x) lazily evaluated for each element x in range . Finally, the last element has the same value as fold!(fun)(seed, range) . The one-argument version cumulativeFold!(fun)(range) works similarly, but it returns the first element unchanged and uses it as seed for the next elements. This function is also known as $(HTTP en.cppreference.com/w/cpp/algorithm/partial_sum, partial_sum), $(HTTP docs.python.org/3/library/itertools.html#itertools.accumulate, accumulate), $(HTTP hackage.haskell.org/package/base-4.8.2.0/docs/Prelude.html#v:scanl, scan), $(HTTP mathworld.wolfram.com/CumulativeSum.html, Cumulative Sum).
