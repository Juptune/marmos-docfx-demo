Module - std.algorithm.mutation

About

This is a submodule of $(MREF std, algorithm). It contains generic mutation algorithms.

Enums

Name Description
SwapStrategy Defines the swapping strategy for algorithms that need to swap elements in a range (such as partition and sort). The strategy concerns the swapping of elements that are not the core concern of the algorithm. For example, consider an algorithm that sorts $(D [ "abc", "b", "aBc" ]) according to toUpper(a) < toUpper(b) . That algorithm might choose to swap the two equivalent strings "abc" and "aBc" . That does not affect the sorting since both ["abc", "aBc", "b" ] and [ "aBc", "abc", "b" ] are valid outcomes.
