Choose one of two ranges at runtime depending on a Boolean condition.

The ranges may be different, but they must have compatible element types (i.e. CommonType must exist for the two element types). The result is a range that offers the weakest capabilities of the two (e.g. ForwardRange if $(D R1) is a random-access range and R2 is a forward range).

Params

condition bool

which range to choose: r1 if true , r2 otherwise

r1 R1

the "true" range

r2 R2

the "false" range

Returns

A range type dependent on R1 and R2 .
