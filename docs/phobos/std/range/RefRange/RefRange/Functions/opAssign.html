auto opAssign( RefRange rhs )

This does not assign the pointer of rhs to this RefRange . Rather it assigns the range pointed to by rhs to the range pointed to by this RefRange . This is because $(I any) operation on a RefRange is the same is if it occurred to the original range. The one exception is when a RefRange is assigned null either directly or because rhs is null . In that case, RefRange no longer refers to the original range but is null .
