About

Given a range of ranges, iterate transversally through the n th element of each of the enclosed ranges. This function is similar to unzip in other languages.

Params

opt

Controls the assumptions the function makes about the lengths

of the ranges rr = An input range of random access ranges Returns: At minimum, an input range. Range primitives such as bidirectionality and random access are given if the element type of rr provides them.
