About

An $(REF_ALTTEXT input range, isInputRange, std,range,primitives) that iterates over the encoded bytes of the given source data.

It will be a $(REF_ALTTEXT forward range, isForwardRange, std,range,primitives) if the underlying data source is at least a forward range.

Note: This struct is not intended to be created in user code directly; use the $(LREF encoder) function instead.
