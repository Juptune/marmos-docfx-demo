max() 

size_t max() @property

Returns the largest allocation size eligible for allocation from the freelist. (If $(D maxSize != chooseAtRuntime), this is simply an alias for maxSize .) All allocation requests for sizes greater than or equal to min and less than or equal to max are rounded to $(D max) and forwarded to the parent allocator. When the block fitting the same constraint gets deallocated, it is put in the freelist with the allocated size assumed to be max .

max(high) 

void max( size_t high ) @property

If FreeList has been instantiated with $(D maxSize == chooseAtRuntime), then the max property is writable. Setting it must precede any allocation.

Params

high size_t

new value for max

Precondition: $(D high >= min), or $(D minSize == chooseAtRuntime) and min has not yet been initialized. Also $(D high >= (void*).sizeof). Also, no allocation has been yet done with this allocator.

Postcondition: $(D max == high)
