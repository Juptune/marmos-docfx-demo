About

A reference counted struct that wraps the dynamic allocator interface. This should be used wherever a uniform type is required for encapsulating various allocator implementations.

Code that defines allocators ultimately implements the $(LREF IAllocator) interface, possibly by using $(LREF CAllocatorImpl) below, and then build a RCIAllocator out of this.

Composition of allocators is not recommended at this level due to inflexibility of dynamic interfaces and inefficiencies caused by cascaded multiple calls. Instead, compose allocators using the static interface defined in $(A std_experimental_allocator_building_blocks.html, std.experimental.allocator.building_blocks ), then adapt the composed allocator to RCIAllocator (possibly by using $(LREF allocatorObject) below).
