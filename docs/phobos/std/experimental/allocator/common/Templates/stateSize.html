std.experimental.allocator.common
template stateSize
{
  // 
  <enum or auto> stateSize = __traits(classInstanceSize, T)
}

About

Returns the size in bytes of the state that needs to be allocated to hold an object of type T . stateSize!T is zero for struct s that are not nested and have no nonstatic member variables.
