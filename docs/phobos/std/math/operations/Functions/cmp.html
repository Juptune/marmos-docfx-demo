cmp(x, y) 

int cmp( T )( T x, T y ) @nogc @trusted nothrow pure

Defines a total order on all floating-point numbers.

The order is defined as follows

$(UL \((LI All numbers in [-\)(INFIN), +$(INFIN)] are ordered the same way as by built-in comparison, with the exception of -0.0, which is less than +0.0;) $(LI If the sign bit is set (that is, it's 'negative'), $(NAN) is less than any number; if the sign bit is not set (it is 'positive'), $(NAN) is greater than any number;) $(LI $(NAN)s of the same sign are ordered by the payload ('negative' ones - in reverse order).) )

Returns

negative value if x precedes y in the order specified above; 0 if x and y are identical, and positive value otherwise.
