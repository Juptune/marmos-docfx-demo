Functions

Name Description
fmod Calculates the remainder from the calculation x/y. Returns: The value of x - i y, where i is the number of times that y can be completely subtracted from x. The result has the same sign as x.
modf Breaks x into an integral part and a fractional part, each of which has the same sign as x. The integral part is stored in i. Returns: The fractional part of x.
remainder Calculate the remainder x REM y, following IEC 60559.
