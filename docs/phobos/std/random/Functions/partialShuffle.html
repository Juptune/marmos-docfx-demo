partialShuffle(r, n, gen) 

Range partialShuffle( Range, RandomGen )( Range r, in size_t n, ref RandomGen gen )

Partially shuffles the elements of r such that upon returning $(D r[0 .. n]) is a random subset of r and is randomly ordered. $(D r[n .. r.length]) will contain the elements not in $(D r[0 .. n]). These will be in an undefined order, but will not be random in the sense that their order after partialShuffle returns will not be independent of their order before partialShuffle was called.

r must be a random-access range with length. n must be less than or equal to r.length . If no RNG is specified, rndGen will be used.

Params

r Range

random-access range whose elements are to be shuffled

n size_t

number of elements of r to shuffle (counting from the beginning); must be less than r.length

gen RandomGen

(optional) random number generator to use; if not specified, defaults to rndGen

n = number of elements of r to shuffle (counting from the beginning); must be less than r.length gen = (optional) random number generator to use; if not specified, defaults to rndGen Returns: The shuffled random-access range.
