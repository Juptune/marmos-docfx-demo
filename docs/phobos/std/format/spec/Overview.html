Module - std.format.spec

About

This is a submodule of $(MREF std, format).

It centers around a struct called $(LREF FormatSpec), which takes a $(MREF_ALTTEXT format string, std,format) and provides tools for parsing this string. Additionally this module contains a function $(LREF singleSpec) which helps treating a single format specifier.
