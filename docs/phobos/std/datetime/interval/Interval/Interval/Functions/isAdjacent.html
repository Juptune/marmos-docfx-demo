Whether the given interval is adjacent to this interval.

Params

interval Interval

The interval to check whether its adjecent to this interval.

Throws

$(REF DateTimeException,std,datetime,date) if either interval is empty.

Example

assert(Interval!Date(Date(1996, 1, 2), Date(2012, 3, 1)).isAdjacent( Interval!Date(Date(2012, 3, 1), Date(2013, 9, 17))));
