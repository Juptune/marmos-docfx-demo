intersects(interval) 

bool intersects( const scope PosInfInterval interval ) nothrow pure

Whether the given interval overlaps this interval.

Always returns true because two intervals going to positive infinity always overlap.

Params

interval PosInfInterval

The interval to check for intersection with this interval.

Example

-------------------- assert(PosInfInterval!Date(Date(1996, 1, 2)).intersects( PosInfInterval!Date(Date(1990, 1, 7))));

assert(PosInfInterval!Date(Date(1996, 1, 2)).intersects( PosInfInterval!Date(Date(1999, 5, 4)))); --------------------
